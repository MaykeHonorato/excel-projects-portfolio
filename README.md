 📊 Portfólio de Projetos em Excel

Bem-vindo ao meu repositório de projetos em Excel!  
Aqui você encontrará soluções desenvolvidas com foco em análise de dados, automação e visualização de informações.

 📁 Estrutura do Repositório


💰 Cofre
Sistema de controle financeiro pessoal desenvolvido em Excel.

**Funcionalidades:**
- Controle de receitas e despesas
- Cálculo automático de saldo
- Indicadores financeiros
- Organização por categorias

 📦 Controle de Importação
Planilha para gestão de produtos importados.

**Funcionalidades:**
- Cadastro de produtos
- Controle de custos (produto, frete, taxas)
- Cálculo de margem de lucro
- Organização de pedidos

📄 Informe de Rendimento
Ferramenta para organização de rendimentos financeiros.

**Funcionalidades:**
- Registro de entradas financeiras
- Apoio para declaração de imposto de renda
- Consolidação de dados
- Visão geral dos ganhos
