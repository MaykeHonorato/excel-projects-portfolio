Controle de Importações – Excel Dashboard
📌 Sobre o Projeto

Desenvolvi este relatório no laboratório da DIO (Digital Innovation One) com o objetivo de criar um dashboard completo para controle de importações de jogos comprados em e-commerce, utilizando códigos de rastreio e tratamento de dados no Excel.

O projeto simula um cenário real de acompanhamento de pedidos, categorizando:

📦 Pedidos que estão para chegar

⚠️ Pedidos com problema

🔄 Pedidos que serão devolvidos

🚚 Pedidos por status (ex: Shipped)

🛠️ Técnicas e Funções Utilizadas

O projeto foi desenvolvido com foco em tratamento de dados, automação e organização dinâmica, utilizando:

🔎 PROC V (Procura Vertical)

Cruzamento de informações entre planilhas

Associação de pedidos com suas respectivas categorias e status

🧹 Tratamento e Limpeza de Texto

Utilizei funções para extrair e padronizar informações contidas dentro de colchetes nos registros:

=SUBSTITUIR(D20;"[";"")
=PROCURAR("[";C20)
=EXT.TEXTO(C20;(PROCURAR("[";C20)+1);(PROCURAR("]";C20)-(PROCURAR("[";C20)+1)))
=SUBSTITUIR(SUBSTITUIR(DIREITA(C21;5);"[";"");"]";"")

📌 Objetivo:

Remover caracteres especiais

Extrair versão do jogo

Separar informações estruturadas dentro de textos não padronizados

📦 Controle de Importações – Dashboard em Excel
📌 Sobre o Projeto

Este projeto foi desenvolvido no laboratório da DIO (Digital Innovation One) com o objetivo de criar um dashboard de controle de importações de jogos adquiridos em e-commerce, utilizando Excel como ferramenta principal de tratamento, organização e análise de dados.

O relatório permite acompanhar pedidos com base em:

📦 Pedidos que estão para chegar

⚠️ Pedidos com problema

🔄 Pedidos que serão devolvidos

🚚 Pedidos organizados por status (ex: Shipped)

O projeto simula um cenário real de controle logístico e organização de pedidos internacionais.

🎯 Objetivo

Demonstrar habilidades em:

Tratamento e limpeza de dados

🔎 Principais Funções Aplicadas
🔹 PROC V (Procura Vertical)

Utilizado para cruzar informações entre a base de dados principal e o painel de controle, permitindo associação automática de dados como status e categorias.

=PROCV(valor_procurado; matriz_tabela; núm_índice_coluna; [procurar_intervalo])
🔹 Extração de Texto entre Colchetes

Aplicação de funções para identificar e extrair informações específicas (como versão do jogo) dentro de textos estruturados com caracteres especiais.

=PROCURAR("[";C20)

=EXT.TEXTO(C20;
(PROCURAR("[";C20)+1);
(PROCURAR("]";C20)-(PROCURAR("[";C20)+1)))

=SUBSTITUIR(D20;"[";"")

Objetivo:

Remover caracteres especiais

Extrair versões como: Normal [XP], Complete [DLX], DLC+

Padronizar dados para análise

🔹 Filtro Dinâmico
=FILTRO(PEDIDOS!$B:$L;PEDIDOS!$H:$H=B49)

Permite:

Filtrar pedidos automaticamente por status

Atualização dinâmica do dashboard

Separação inteligente de dados

🔹 Tratamento de Texto (Versões)
=SUBSTITUIR(SUBSTITUIR(DIREITA(C21;5);"[";"");"]";"")

Usado para:

Extrair códigos finais de versão

Remover colchetes

Padronizar informação de forma automatizada
